Explicación Profunda - Pregunta 4 del FE Mechanical

Solución de una Ecuación Diferencial Ordinaria de Primer Orden

Pregunta Original

4. Which of the following is the general solution to the differential equation and boundary condition shown below?

dy/dt + 5y = 0; y(0) = 1

Opciones: A. e5t   B. e-5t   C. e√(-5t)   D. 5e-5t

Solución Paso a Paso

1

Identificar el tipo de ecuación diferencial

La ecuación dada es:

dy/dt + 5y = 0

Esta es una ecuación diferencial ordinaria (EDO) de primer orden porque:

  • Contiene solo derivadas ordinarias (no parciales)
  • La derivada más alta es de primer orden (dy/dt)
  • Es lineal porque y y dy/dt aparecen solo a la primera potencia
  • Es homogénea porque el término independiente es 0
  • Tiene coeficientes constantes (5 es constante)
2

Resolver usando el método de separación de variables

Método 1: Separación de variables

Reescribimos la ecuación:

dy/dt = -5y

Separamos variables (y a un lado, t al otro):

dy/y = -5 dt

Integramos ambos lados:

∫ dy/y = ∫ -5 dt
ln|y| = -5t + C₁

Aplicamos exponencial a ambos lados:

y = e-5t + C₁ = eC₁ × e-5t

Llamamos C = eC₁ (constante arbitraria):

y(t) = C e-5t
3

Resolver usando el método del factor integrante

Método 2: Factor integrante

Para una EDO lineal de primer orden en forma estándar:

dy/dt + P(t)y = Q(t)

En nuestro caso: P(t) = 5, Q(t) = 0

El factor integrante es:

μ(t) = e∫P(t)dt = e∫5dt = e5t

Multiplicamos la ecuación original por el factor integrante:

e5t dy/dt + 5e5t y = 0

El lado izquierdo es la derivada del producto:

d/dt [y e5t] = 0

Integramos ambos lados con respecto a t:

y e5t = C

Despejamos y:

y(t) = C e-5t

¡Obtenemos el mismo resultado!

4

Aplicar la condición inicial

Tenemos la condición inicial: y(0) = 1

Sustituimos t = 0 en nuestra solución general:

y(0) = C e-5×0 = C e0 = C × 1 = C

Pero y(0) = 1, por lo tanto:

C = 1

La solución particular que satisface la condición inicial es:

y(t) = 1 × e-5t = e-5t

5

Verificar las opciones dadas

Comparamos nuestra solución con las opciones:

Opción Función ¿Satisface la EDO? ¿Satisface y(0)=1? Comentario
A e5t No Sí (e0=1) Derivada: 5e5t. dy/dt + 5y = 5e5t + 5e5t = 10e5t ≠ 0
B e-5t Sí Sí Derivada: -5e-5t. dy/dt + 5y = -5e-5t + 5e-5t = 0 ✓
C e√(-5t) No Sí (e0=1) Función compleja para t>0. No es solución de EDO con coeficientes reales.
D 5e-5t Sí No (5e0=5≠1) Satisface la EDO pero no la condición inicial.

Conclusión: Solo la opción B satisface tanto la ecuación diferencial como la condición inicial.

Respuesta Correcta
e-5t (Opción B)

La solución de dy/dt + 5y = 0 con y(0) = 1 es y(t) = e-5t

Conceptos Clave Involucrados

  • Ecuaciones Diferenciales Ordinarias (EDOs): Ecuaciones que contienen una o más derivadas de una función desconocida con respecto a una sola variable independiente.
  • EDO de Primer Orden: Contiene solo la primera derivada de la función desconocida.
  • EDO Lineal: La función desconocida y sus derivadas aparecen solo a la primera potencia y no se multiplican entre sí.
  • EDO Homogénea: Todos los términos contienen la función desconocida o sus derivadas (término independiente = 0).
  • Solución General: Contiene constantes arbitrarias (en este caso, C).
  • Solución Particular: Se obtiene de la solución general aplicando condiciones iniciales o de frontera.
  • Constante de Tiempo (τ): Para y' + ay = 0, τ = 1/a. Representa el tiempo en que la solución decae a 1/e ≈ 36.8% de su valor inicial.

Aplicaciones en Ingeniería Mecánica

  • Decaimiento de Temperatura (Ley de Enfriamiento de Newton): dT/dt = -k(T - Tₐ), que se reduce a esta forma cuando Tₐ = 0.
  • Descarga de Capacitores en Circuitos RC: V' + (1/RC)V = 0, donde V es el voltaje en el capacitor.
  • Amortiguamiento Viscoso (sin inercia): F = -cv, que para un sistema masa-resorte-amortiguador se convierte en mx'' + cx' + kx = 0.
  • Reacciones Químicas de Primer Orden: dC/dt = -kC, donde C es la concentración y k es la constante de velocidad.
  • Dinámica de Sistemas de Primer Orden: Muchos sistemas físicos se modelan como sistemas de primer orden cuando la inercia es despreciable.
6

Interpretación física y constante de tiempo

Para la ecuación dy/dt + 5y = 0, podemos reescribirla como:

dy/dt = -5y

Esto significa que la tasa de cambio de y es proporcional a y misma, con constante de proporcionalidad -5.

La constante de tiempo τ es:

τ = 1/5 = 0.2 unidades de tiempo

Interpretación física:

  • En t = τ = 0.2, y(0.2) = e-5×0.2 = e-1 ≈ 0.368 (36.8% del valor inicial)
  • En t = 2τ = 0.4, y(0.4) = e-2 ≈ 0.135 (13.5% del valor inicial)
  • En t = 3τ = 0.6, y(0.6) = e-3 ≈ 0.050 (5.0% del valor inicial)
  • En t = 5τ = 1.0, y(1.0) = e-5 ≈ 0.0067 (0.67% del valor inicial)

Aplicación práctica: Si y representa la temperatura de un objeto que se enfría en un ambiente a 0°C, después de 0.2 unidades de tiempo, la temperatura habrá descendido al 36.8% de su valor inicial.

7

Método de la ecuación característica (para EDOs lineales con coeficientes constantes)

Método 3: Ecuación característica

Para una EDO lineal homogénea con coeficientes constantes:

dy/dt + ay = 0

Proponemos una solución de la forma y = ert, donde r es una constante a determinar.

Sustituyendo en la EDO:

d(ert)/dt + a(ert) = r ert + a ert = (r + a)ert = 0

Como ert ≠ 0, tenemos la ecuación característica:

r + a = 0

Resolviendo: r = -a

Para nuestro caso: a = 5, por lo tanto r = -5

La solución general es:

y(t) = C ert = C e-5t

Referencia al FE Reference Handbook: Esta pregunta se refiere a la sección de Mathematics, específicamente a Ordinary Differential Equations, donde se encuentran los métodos para resolver EDOs de primer orden.

La solución de dy/dt + ay = 0 es y = Ce-at, y con condición inicial y(0) = y₀, la solución es y = y₀e-at.
