Explicación Profunda - Pregunta 5 del FE Mechanical

Cálculo de un vector unitario perpendicular a un plano definido por dos vectores

Pregunta Original

5. Which of the following is a unit vector perpendicular to the plane determined by the vectors A = 2i + 4j and B = i + j - k?

Opciones:

A. -2i + j - k

B. (1/√5)(i + 2j)

C. (1/√6)(-2i + j - k)

D. (1/√6)(-2i - j - k)

Solución Paso a Paso

1

Entender el problema

Nos piden encontrar un vector unitario (vector de magnitud 1) que sea perpendicular al plano determinado por dos vectores dados:

A = 2i + 4j + 0k
B = 1i + 1j - 1k

En notación de componentes:

Vector A
Ax = 2
Ay = 4
Az = 0
Vector B
Bx = 1
By = 1
Bz = -1

Un vector perpendicular a ambos A y B es su producto cruz (producto vectorial). Luego, normalizamos ese vector para obtener un vector unitario.

2

Calcular el producto cruz A × B

El producto cruz de dos vectores A = (Ax, Ay, Az) y B = (Bx, By, Bz) es:

A × B = (AyBz - AzBy)i - (AxBz - AzBx)j + (AxBy - AyBx)k

O usando notación de determinante:

Cálculo por determinante:
i j k
Ax = 2 Ay = 4 Az = 0
Bx = 1 By = 1 Bz = -1

Calculamos cada componente:

Componente i: AyBz - AzBy = (4)(-1) - (0)(1) = -4 - 0 = -4
Componente j: -(AxBz - AzBx) = -[(2)(-1) - (0)(1)] = -[-2 - 0] = -(-2) = 2
Componente k: AxBy - AyBx = (2)(1) - (4)(1) = 2 - 4 = -2

Por lo tanto:

A × B = -4i + 2j - 2k
3

Calcular la magnitud del vector producto cruz

Para normalizar el vector, necesitamos su magnitud:

|A × B| = √[(-4)² + (2)² + (-2)²] = √[16 + 4 + 4] = √24 = √(4×6) = 2√6

Verificación de cálculo:

  • (-4)² = 16
  • (2)² = 4
  • (-2)² = 4
  • Suma: 16 + 4 + 4 = 24
  • √24 = √(4×6) = 2√6 ≈ 4.899
4

Normalizar el vector para obtener un vector unitario

Un vector unitario en la dirección de A × B se obtiene dividiendo cada componente por la magnitud:

u = (A × B) / |A × B| = (-4i + 2j - 2k) / (2√6)

Simplificamos factorizando 2 en el numerador:

u = [2(-2i + j - k)] / (2√6) = (-2i + j - k) / √6

Verificamos que es unitario calculando su magnitud:

|u| = √[(-2/√6)² + (1/√6)² + (-1/√6)²] = √[(4/6) + (1/6) + (1/6)] = √(6/6) = √1 = 1

Vector unitario resultante: u = (1/√6)(-2i + j - k)

Respuesta Correcta
(1/√6)(-2i + j - k) (Opción C)

El vector unitario perpendicular al plano determinado por A = 2i + 4j y B = i + j - k

Conceptos Clave Involucrados

  • Producto Cruz (Producto Vectorial): Operación entre dos vectores en el espacio tridimensional que produce un vector perpendicular a ambos. Si A y B son vectores, entonces A × B es perpendicular a A y a B.
  • Vector Unitario: Vector de magnitud 1. Se obtiene dividiendo un vector por su magnitud: û = v/|v|.
  • Normalización: Proceso de convertir un vector en un vector unitario manteniendo su dirección.
  • Magnitud de un Vector: |v| = √(vₓ² + vᵧ² + v₂²). Representa la longitud del vector.
  • Plano Definido por Dos Vectores: Dos vectores no paralelos en el espacio tridimensional definen un plano único. Cualquier vector perpendicular a ambos será perpendicular al plano.
  • Notación i, j, k: Vectores unitarios en las direcciones x, y, z respectivamente.

Aplicaciones en Ingeniería Mecánica

  • Torque (Momento de Fuerza): τ = r × F, donde r es el vector posición y F es la fuerza. El torque es perpendicular al plano que contiene r y F.
  • Velocidad Angular: En rotación, la velocidad angular ω es perpendicular al plano de rotación.
  • Área de Paralelogramos: El área del paralelogramo formado por dos vectores A y B es |A × B|.
  • Normal a Superficies: En análisis de esfuerzos y diseño de superficies, se usan vectores normales para definir orientaciones.
  • Análisis Estructural: Determinación de fuerzas normales a planos en estructuras tridimensionales.
  • Dinámica de Cuerpos Rígidos: El momento angular L = r × p es perpendicular al plano que contiene la posición y el momento lineal.
6

Verificación mediante producto punto

Podemos verificar que nuestro vector unitario es realmente perpendicular a A y B calculando el producto punto:

Producto punto con A:

u · A = (-2/√6)(2) + (1/√6)(4) + (-1/√6)(0) = (-4/√6) + (4/√6) + 0 = 0

Producto punto con B:

u · B = (-2/√6)(1) + (1/√6)(1) + (-1/√6)(-1) = (-2/√6) + (1/√6) + (1/√6) = 0

Como u · A = 0 y u · B = 0, el vector u es perpendicular a ambos A y B, y por lo tanto perpendicular al plano que definen.

7

Propiedades importantes del producto cruz

Propiedades del Producto Cruz:
  • Anticonmutatividad: A × B = - (B × A)
  • Distributividad: A × (B + C) = A × B + A × C
  • Asociatividad con escalares: (cA) × B = c(A × B) = A × (cB)
  • Producto cruz de vectores paralelos: Si A es paralelo a B, entonces A × B = 0
  • Magnitud: |A × B| = |A||B|sinθ, donde θ es el ángulo entre A y B
  • Dirección: A × B es perpendicular a ambos A y B, determinado por la regla de la mano derecha

En nuestro problema: |A| = √(2²+4²+0²) = √20 = 2√5 ≈ 4.47, |B| = √(1²+1²+(-1)²) = √3 ≈ 1.73

El ángulo θ entre A y B se puede encontrar usando: sinθ = |A×B|/(|A||B|) = (2√6)/(2√5×√3) = √6/√15 = √(2/5) ≈ 0.632
