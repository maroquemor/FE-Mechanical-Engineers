Explicación Profunda - Pregunta 3 del FE Mechanical

Estimación del área bajo una curva usando la Regla del Trapecio

Pregunta Original

3. Suppose f(t) = t². The area under the curve for 0 ≤ t ≤ 2, estimated by using the trapezoidal rule with Δt = 0.5, is most nearly:

Opciones: A. 4.00   B. 2.75   C. 2.67   D. 1.33

Solución Paso a Paso

1

Entender el problema

Nos piden estimar el área bajo la curva de la función f(t) = t² en el intervalo [0, 2] usando la Regla del Trapecio con un ancho de intervalo Δt = 0.5.

El área bajo la curva f(t) desde t = a hasta t = b está dada por la integral definida:

A = ∫ab f(t) dt

En nuestro caso específico: f(t) = t², a = 0, b = 2.

La Regla del Trapecio es un método de integración numérica que aproxima el área bajo una curva dividiéndola en trapecios.

2

Configurar la partición del intervalo

Con Δt = 0.5 y el intervalo [0, 2], necesitamos determinar los puntos de la partición:

t₀ = 0, t₁ = 0.5, t₂ = 1.0, t₃ = 1.5, t₄ = 2.0

Esto nos da 4 subintervalos de igual ancho (0.5 cada uno).

3

Calcular los valores de la función en cada punto

Evaluamos f(t) = t² en cada punto de la partición:

i tᵢ f(tᵢ) = tᵢ² Factor en la fórmula
0 0.0 0² = 0.00 1 × f(t₀)
1 0.5 0.5² = 0.25 2 × f(t₁)
2 1.0 1.0² = 1.00 2 × f(t₂)
3 1.5 1.5² = 2.25 2 × f(t₃)
4 2.0 2.0² = 4.00 1 × f(t₄)
4

Aplicar la fórmula de la Regla del Trapecio

La fórmula general de la Regla del Trapecio para n subintervalos es:

∫ab f(t) dt ≈ (Δt/2) × [f(t₀) + 2f(t₁) + 2f(t₂) + ... + 2f(tₙ₋₁) + f(tₙ)]

Para nuestro caso específico (n = 4 subintervalos):

A ≈ (0.5/2) × [f(0) + 2f(0.5) + 2f(1.0) + 2f(1.5) + f(2.0)]

A ≈ 0.25 × [0 + 2(0.25) + 2(1.0) + 2(2.25) + 4]

A ≈ 0.25 × [0 + 0.5 + 2.0 + 4.5 + 4]

A ≈ 0.25 × [11.0]

A ≈ 2.75

5

Verificar la solución

Verificación analítica exacta: Podemos calcular el área exacta mediante integración:

Aexacta = ∫02 t² dt = [t³/3]02 = (8/3) - 0 = 2.666...

Comparación: Nuestra aproximación de 2.75 es ligeramente mayor que el valor exacto de 2.666...

Error porcentual:

Error = |2.75 - 2.666...| / 2.666... × 100% ≈ 3.125%

¡Esto es una excelente aproximación con solo 4 subintervalos!

Respuesta Correcta
2.75 (Opción B)

El área estimada bajo la curva f(t) = t² de 0 a 2 usando la Regla del Trapecio con Δt = 0.5 es 2.75

Conceptos Clave Involucrados

  • Integración Numérica: Métodos para aproximar el valor de integrales definidas cuando no se puede o no es práctico encontrar una solución analítica exacta.
  • Regla del Trapecio: Un método de integración numérica que aproxima el área bajo una curva como la suma de áreas de trapecios.
  • Error de Aproximación: La diferencia entre el valor aproximado y el valor exacto. Para la Regla del Trapecio, el error es proporcional a (Δt)².
  • Particionamiento del Intervalo: Dividir el intervalo de integración en subintervalos de igual ancho para aplicar métodos numéricos.
  • Función Cuadrática: f(t) = t² es una parábola, cuya integral exacta en [0, 2] es 8/3 ≈ 2.6667.

Aplicaciones en Ingeniería Mecánica

  • Análisis de Datos Experimentales: Cuando se tienen mediciones discretas de una variable, la Regla del Trapecio se usa para aproximar integrales (como trabajo, energía, desplazamiento).
  • Cálculo de Propiedades de Materiales: Determinar áreas bajo curvas esfuerzo-deformación para calcular energía de deformación.
  • Análisis de Señales: Integración numérica de señales de aceleración para obtener velocidad y desplazamiento.
  • Diseño de Curvas: Calcular áreas y volúmenes de componentes con formas complejas definidas por puntos discretos.
  • Simulación Numérica: En métodos como el MEF (Método de Elementos Finitos), se usan técnicas de integración numérica para evaluar integrales sobre elementos.
6

Comparación con otros métodos de integración numérica

Regla del Trapecio

Fórmula: (Δt/2)[f₀ + 2∑fᵢ + fₙ]

Nuestro resultado: 2.75

Error: +0.0833 (3.1%)

Ventaja: Simple, buena para funciones aproximadamente lineales por tramos.

Regla de Simpson (1/3)

Fórmula: (Δt/3)[f₀ + 4f₁ + 2f₂ + 4f₃ + f₄]

Resultado: 2.6667 (exacto para polinomios cúbicos)

Error: 0 (para f(t)=t²)

Ventaja: Más precisa para funciones suaves.

Regla del Punto Medio

Fórmula: Δt∑f((tᵢ+tᵢ₊₁)/2)

Resultado: 2.625

Error: -0.0417 (1.56%)

Ventaja: A veces más precisa que el trapecio con el mismo número de evaluaciones.

7

Análisis del error y convergencia

Para la Regla del Trapecio, el error de truncamiento está dado por:

E = -(b-a) × (Δt)² × f''(ξ) / 12

Donde ξ es algún punto en [a, b]. Para f(t) = t²:

  • f''(t) = 2 (constante)
  • b - a = 2
  • Δt = 0.5
  • Error teórico máximo: E = -(2) × (0.5)² × 2 / 12 = -0.0833

Nuestro error real fue +0.0833, lo que coincide en magnitud con la predicción teórica.

Nota importante: La Regla del Trapecio sobreestima el área para funciones convexas (f''(t) > 0) como t², y subestima para funciones cóncavas (f''(t) < 0).
